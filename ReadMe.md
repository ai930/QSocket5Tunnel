QSocket5Tunnel 
-----------------------------
    是一个基于Qt的QTcpSocket和QTcpServer写的一个socket5代理的数据中转软件。
    分为客户端和服务端两部分，客户端负责本地建立socket5代理，服务端负责负责请求代理客户端需要请求的数据。
    其中客户端和服务端用一个TCP长链接链接。
    其中不用socket5的认证，自己单独的认证
